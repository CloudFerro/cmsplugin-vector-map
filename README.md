# cmsplugin-vector-map

### Requirements

django

django-cms

djangocms-text-ckeditor

### Installation

Add cmsplugin_vector_map to INSTALLED APPS:

```python
INSTALLED_APPS = (
        ...
        'cmsplugin_vector_map',
        ...
    )
```

That's it!
Plugin is ready to use in Django CMS's placeholders
